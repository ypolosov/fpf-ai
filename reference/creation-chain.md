# Creation Chain

## Concept
A creation chain is a sequence of creating systems where each creates value
for the next. Each creating system is universal — it can serve any target.

## fpf-ai's Role: Thinking Layer
fpf-ai provides first-principles thinking capabilities:
- **Boundary analysis** — identify and evaluate component boundaries
- **Trust calibration** — classify knowledge as Fact / Guess / Rumor
- **Part-whole decomposition** — verify how parts compose into wholes
- **Evolution readiness** — assess explore/exploit balance
- **Terminology precision** — build unambiguous domain glossaries
- **Characteristics chains** — trace ideas to measurable metrics

These capabilities are applicable to any system, including other creating
systems (methodology tools, knowledge platforms, etc.).

## Important
fpf-ai is fully universal. This context is informational only.
